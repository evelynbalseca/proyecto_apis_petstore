# Proyecto: serenity_apis_petstore

## Tecnologías usadas:
- IntelliJ IDEA 2024.1.4 - Build #IC-241.18034.62
- Java vs 17.0.4
- Maven vs 4.0.0
- JUnit vs 5
- Karate vs 1.4.1
- SO Windows 11 Pro


## Pasos de ejecución:
1. Descargar el proyecto serenity_apis_petstore del github: https://github.com/evelynbalseca/proyecto_apis_petstore.git por medio del cmd con el comando: git clone https://github.com/evelynbalseca/proyecto_apis_petstore.git
2. Importar el proyecto serenity_apis_petstore descargado en IntelliJ.
3. Ejecutar el archivo RunPetKarate.java que se encuntra en src/test/java/com/petstore/configs.
4. Al finalizar las pruebas revisar el reporte en la consola en la seccion "HTML report: (paste into browser to view)" y dar clic en el archivo.html generado;o se puede descargar en la ruta file:..../target/karate-reports/karate-summary.html.
